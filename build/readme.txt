Mit ein paar einfachen Klicks binden Sie Ihre BOTTIMMO Widgets in Ihrer WordPress Website ein.

== Description ==

Als BOTTIMMO Kunde haben Sie die Möglichkeit die verschiedensten Widgets, die BOTTIMMO Ihnen zur Verfügungs stellt als
sogenannte Iframes in Ihrer Website einzubinden.

Nähere Infos dazu, finden Sie in Ihrer BOTTIMMO App.

Dieses kostenlose WordPress Plugin ermöglicht Ihnen die Einbindung der Widgets mit nur wenigen Klicks. Alles was Sie dazu benötigen ist
Ihr individuelles Firmenkürzel, dass BOTTIMMO Ihnen erstellt hat.

== Frequently Asked Questions ==

= Wo finde ich mein individuelles Firmenkürzel (Slug)? =

Ihr Firmenkürzel (auch Slug genannt) ist der erste Teil Ihrer Landingpage Domain nach dem "https://" und dem ersten Punkt.

Z.B. https://max-makler.immowissen.org (hier ist "max-makler" das Firmenkürzel)

= Wo kann ich das Firmenküzel setzen? =

Nach erfolgreicher Installation des "BOTTIMMO Widget" Plugins, erhalten Sie in Ihrer Admin-Navigation unter dem Menüpunkt "Einstellungen" 
einen weiteren Menüpunkt namens "BOTTIMMO Widget". Über diesen Link gelangen Sie zur Eingabemaske für das Firmenkürzel.

= Wie kann ich ein Widget einbinden? =

Voraussetzung hierfür ist, dass Ihre WordPress Installation sogenannte "Gutenberg Blocks" unterstützt. Dies ist ab WordPress Version 5.9 standardmäßig der Fall.

Navigieren Sie zu der Seite oder dem Post, in dem Sie ein oder mehrere BOTTIMMO Widgets einbinden wollen. 

Erstellen Sie einen neuen Block (build/assets/screenshot-2.png)

Suchen Sie nach dem BOTTIMMO Widget Plugin (build/assets/screenshot-3.png)

Nach Auswahl des Blocks, folgen Sie den Anweisungen im Plugin (build/assets/screenshot-4.png)
